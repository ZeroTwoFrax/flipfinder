# FlipFinder

FlipFinder is a mobile app that helps users quickly determine whether an item is worth reselling for profit.  
Users enter the purchase price and the app estimates market value and potential profit.

The goal is to help resellers, thrift shoppers, and marketplace flippers make smarter buying decisions.

---

## Features

- Clean Apple-style mobile interface
- Profit estimation calculator
- Market value estimate
- Flip rating system
- Built with React Native + Expo
- Fast mobile testing with Expo Go

Upcoming features:
- Scan items with camera
- Real resale data from marketplaces
- Deal alerts
- Saved flips dashboard
- Profit tracking

---

## Tech Stack

- React Native
- Expo
- JavaScript / TypeScript
- Node.js

Project created using the Expo development platform.
